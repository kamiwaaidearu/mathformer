# Mathformer

A math platformer game for kids! Guide a hungry kitten across platforms to solve math problems and feed it tasty treats.

## How to Play

- **Arrow Keys** — Move left/right
- **Space** — Jump
- **Up/Down Arrows** — Navigate the level select menu

Land on the platform with the correct answer to feed the kitten. Get all 10 right to win!

## Levels

- **Doubles** — Addition with matching numbers (1+1, 2+2, 3+3...)
- **Make 10** — Find the missing number that adds up to 10 (1 + ? = 10, 3 + ? = 10...)
- **Tens & Ones** — Place value problems (3 tens + 5 ones = ?, tens digit of 47 = ?...)

## Getting Started

Requires [Node.js](https://nodejs.org/) (v18+).

```sh
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build

```sh
npm run build
npm run preview
```

## Tech

Built with TypeScript, HTML Canvas, and Vite. No game engine — just raw canvas rendering with Mario-style platformer physics.
